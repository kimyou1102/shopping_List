# Shopping List
강의의 sloution을 보기 전 스스로 구현해 본, 쇼핑목록을 작성하고 삭제할 수 있는 매우 간단한 프로젝트 입니다.

# 기술 스택
HTML, CSS, JAVASCRIPT
